Project Overview

IBG(Institutional Banking Group) I.Serve is the application which works for corporate banking. This application is used by customer service operations teams who directly are in contact with corporate customers for their queries resolving.

I.Serve is a one-stop solution to have enhanced SR Dashboard and search functions, and to have customer profile in a single screen, and to have manual authentication with smart answers and easy navigation to products and transaction details.

It helps the CSO team to answer customer's queries very fast. It saves a lot of time by not toggling many systems to gather customer information. This application is for Service Requests Management.

Features of the application

CSO team can create new SR as draft with user query and assign the SR to themself or their colleagues.

This application helps the CSO team to collect customer information at once in order to raise SR easily.

It has enhanced SR Dashboard and Search function which helps the CSO team to see an overview of individual and team SR Dashboard. It also helps to search customer information by their account number which provides a customer profile in a single screen.

Easy access to SR Inbox for SR with Open or Draft Status. And detailed view of Open and Draft SR upon click of SR.

It allows the creation of SR, Modify SR, Update SR, and Clone SR.

It allows you to send SMS and search about SR and error payments.

It provides an overview of transaction details at any point of time.and Fixed Deposit, Loans Details, Debit and Credit cards Informations, and key customer contacts detail.

It allows managers to download reports of SR List and various other reports.
