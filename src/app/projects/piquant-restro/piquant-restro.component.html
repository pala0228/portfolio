Project Overview

Piquant Restro is a website collecting restaurants from different places and providing customers to view them all and place their favourite food items with a dummy stripe payment option(card no- 4242 4242 4242 4242).

They can also view their placed orders and download an invoice of orders.

Features of the application

Users/Admin can sign up, and login with credentials.

Users/Admin can reset their password with OTP receiving on their registered mobile number.

Users/Admin can update their profile information.

Admin can only add/edit/delete restaurant and food items accordingly.

Users can view all available restaurants and place an order with a dummy stripe payment gateway.

Admin can view only restaurants added by them and place an order with dummy stripe payment gateway

Users/Admin can download the invoice in pdf format for their placed orders.

Users will be provided an alert if the page has been idle for 10 minutes, If the user does not click on 'Continue' within 5 minutes of time then the user will be auto logged out from the application.

Super Admin do have all the privileges which users and admin have.
